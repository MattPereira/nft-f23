# Cast practice with anvil chain

step 1: start anvil with `make anvil` and then deploy to anvil with `make deployMood`

### Mint mood nft using cast
cast send <nft_smart_contract_address> "mintNft()" --private-key $ANVIL_PK --rpc-url $ANVIL_RPC_URL

### Flip mood using cast
cast send 0x5FbDB2315678afecb367f032d93F642f64180aa3 "flipMood(uint256)" 0 --private-key $ANVIL_PK --rpc-url $ANVIL_RPC_URL
